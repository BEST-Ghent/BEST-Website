Discover VS Code

VS Code is an open source code editor developed by Microsoft.
You are of course free to use any code editor you fancy, but if this is your first time managing any kind of codebase, it might be wise to start here.

Downloading/Installing

  1. First, download the VS Code installer from the official download page.
  2. Next, run the installer. The default options are usually quite good, except that they don't generate a desktop icon, so make sure to tick that box when you see it.
  3. VS Code has built in support for source control using git, but uses the git installation on your machine, so you need to have git installed to use these features.

Live Server extension

When you make a change to the code of the website, you'll probably want to see how that change affects the website itself. One way to do this (but please don't), is to make a change, push it to github and see what it looks like on the official, public site.

If you'd work like this, you would waste a lot of time and generate a lot of commits. Too many pointless commits will clutter the commit history and make it difficult to understand what changes where made when and for what reason. More importantly you would risk publishing some kind of bug or maybe private information to the public website that is accesible by everyone.

So what do you do? Luckily for us, VS Code comes with a deluge of extensions. One of them is called Live Server. Live Server will open a locally hosted version of your code in your browser and automatically updates this version when you hit save.

This means you can make any change, hit ctrl+s and instantly see what this change will look like in the final product. All from the comfort of your own machine! (works offline too!)

To install this extension, go to the extensions tab in VS Code (ctrl+shift+X), search for Live Server and install the one made by Ritwick Dey. There is also an extension called Live Preview, made by Microsoft, but at the time of writing, this is still under development.

Once installed, a Go Live button will appear in the bottom right corner of your VS Code window. When you go live, it will try to open whatever file you have open at that moment. If that is a html file, it will go to that page, else it will open the homepage (the root index.html).
