Git & VS Code setup

To make changes to the website, you'll need to get a code editor (unless you feel more comfortable using notepad of course). Visual Studio Code is a great code editor that provides a lot of features and has a vast library of free extensions.
Get started with VS Code

Now that you can edit the site, you still need to get your hands on its code. Cloning/pulling (downloading) and pushing (uploading) changes to github can be done using the version control system called git. VS Code provides a great interface to git, but you are free to use any git client you like. Personally, I like to use fork.
Get started using git

How do we host our site?
